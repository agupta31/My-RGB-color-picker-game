Welcome to my own RGB color picker game.This is fairly simple game of guessing a random color from an array of colors shown on the page. The RGB color model is an additive color model in which red,green and blue light are added together in various ways to reproduce a broad array of colors.The page displays an array of random colors out of which one color will correspond to the RGB combination displayed at the top .Your job is to make a calculated guess as to which color represents the right combination and select that color.Yes.It is that simple!!Hope you enjoy playing this game.
